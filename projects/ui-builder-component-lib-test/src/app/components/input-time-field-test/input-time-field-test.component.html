Timepicker

TimePicker is a component allowing users to select times using either manual entry or a dropdown.
Demo
Required Imports
import { InputTimeFieldModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description Default
Input startDate The date from which the time slots are generated. two days from today
Input interval The interval in minutes between time slots. 30 minutes
Input displaySlotTime Whether to display the time difference between slots and the starting time. n/a
Input fromTime The starting time used to generate slots relative to this value. n/a
Input format The time format for displaying time slots, such as 'hh:mm aa'. hh:mm
Input label The label for the input field. n/a
Input readonly Disables the input if true. n/a
Input showIcon Determines if an icon is shown beside the input field. n/a
Input disabled Disables the component entirely if true. n/a
Output focusEvent Emits a boolean value when the input field gains or loses focus. n/a
Public timeSlots An array of generated time slots based on the provided start date and interval. n/a
Public filteredTimeSlots The filtered list of time slots used for display in the dropdown. n/a
Public control A reactive form control used to manage the selected time slot. n/a
Public focused A flag indicating whether the input field is currently focused. n/a
