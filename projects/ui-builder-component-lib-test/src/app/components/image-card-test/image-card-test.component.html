Input and Output Properties
Type Name Description Data type Default
Input imageUrl URL of the image to display. n/a n/a
Input imageLabel Label for the image. n/a n/a
Input maxNumOfPhotos Maximum number of photos allowed. n/a undefined
Input loading Indicates whether the component is loading. n/a n/a
Input disabled Disables the component when set to true. n/a n/a
Input enableDelete Enables or disables the delete option. n/a true
Input enableCropper Enables or disables the image cropper. n/a n/a
Input imageFormats Array of allowed image formats. n/a all available formats
Input cropperModalViewRef Reference to the view container for the cropper modal. n/a n/a
Input enableCompression Enables or disables image compression. n/a true
Input maxImageSizeInBytes Maximum size of the image in bytes. n/a 2MB
Input maxImageCompressSizeInBytes Maximum size of compressed image in bytes. n/a 700KB
Input photoAspectRatio Aspect ratio of the photo. n/a 1 (square)
Input photos Array of photos. ImageModel n/a
Input disableUploadOnClick Disables the upload action on click. n/a false
Input faceDetector Function to detect faces in images. n/a n/a
Input nsfwChecker Function to check for NSFW content in images. n/a n/a
Input enableNsfwCheck Enables NSFW checks on the images. n/a n/a
Input enableFaceCheck Enables face detection on the images. n/a n/a
Output photosUpload Event emitted when photos are uploaded successfully. ImageModel n/a
Output photoUploadFail Event emitted when photo upload fails. string n/a
Output dupPhotosDetect Event emitted when duplicate photos are detected. number n/a
Output unsupportedFormatsDetect Event emitted when unsupported formats are detected. string n/a
Output maxUploadLimitExceeded Event emitted when the upload limit is exceeded. null n/a
