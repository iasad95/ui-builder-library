MultiField
MultiField is an editable list component where each row can consist of one or more fields of various types.
Demo
Required Imports
import { MultiFieldModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description
Input fieldName The name of the field.
Input config Configuration object for form section variables, defining field settings.
Input formFieldControl The form control associated with this component.
Input newRowText Text to display on the button for adding a new row.
Input useTooltips Boolean to enable or disable tooltips for fields.
Input disabled Determines if the entire component is disabled.
Input rowDeleteConfirmationObsv An observable that tracks row deletion confirmation actions.
Input maxLength The maximum length allowed for the field's input value.
Input currencySymbol Defines the currency symbol to use in currency input fields.
Input errorMessage Custom error message to display if validation fails.
Input editDisabled Boolean to disable row editing functionality.
Output rowDelete Event emitted when a row is deleted, sending the field name and row index.
Output rowEdit Event emitted when a row is being edited, containing form controls and the configuration object.
Output valueChanges Event emitted when there are value changes in the fields, passing the updated form control and configuration.
