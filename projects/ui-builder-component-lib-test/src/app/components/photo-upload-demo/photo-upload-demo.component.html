Photo Upload Component

A simple tool for quickly uploading photos in your application.
Demo
Required Imports
import { PhotoUploadModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description
Input maxNumOfPhotos The maximum number of photos that can be uploaded.
Input photos An array of uploaded photos represented by ImageModel.
Input slotsPerRow The number of photo slots to display per row.
Input gridGap The gap (in pixels) between the photo slots.
Input enableDragAndDrop Enables drag-and-drop functionality for uploading images.
Input infoIcon The icon used to display help or information about image uploading.
Input largeFileUploadWarningParams Settings for warning the user when uploading large files.
Input photoAspectRatio The aspect ratio of the photos being uploaded.
Input enableNsfwCheck Enables checking if an uploaded image contains NSFW content.
Input enableFaceCheck Enables checking if an uploaded image contains a face.
Input faceDetector A function to detect faces in the uploaded image.
Input nsfwChecker A function to check for NSFW content in the uploaded image.
Input compressRatio Maximum scale factor as a percentage.
Input compressQuality Image quality factor as a percentage.
Output photosUpload Emits an array of ImageModel whenever photos are uploaded.
Output photosDelete Emits an array of ImageModel whenever a photo is deleted.
Output photoUploadFail Emits the file name if the photo upload fails.
Output dupPhotosDetect Emits the index of the duplicate photo if detected during upload.
Output unsupportedFormatsDetect Emits a list of unsupported file formats detected during upload.
Output maxUploadLimitExceeded Emits a notification when the maximum upload limit is exceeded.
