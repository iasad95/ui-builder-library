Progressbar

A progress bar component that visually indicates the completion status of a task or process, providing users with a clear and simple way to track ongoing activities.
Demo
Required Imports
import { ProgressBarModule } from 'ui-builder-component-lib';

Input progBarTotalSteps The total number of steps in the progress bar.
