Toggle

A custom toggle button component that allows users to toggle a state on or off. The state of the toggle button is controlled by a boolean property and a method is invoked whenever a user clicks the button to change the state.

Input and Output Properties
Type Name Description
Input disabled Determines if the toggle button is disabled, preventing user interaction.
Input toggleState The current state of the toggle button, which can be either true (on) or false (off).
Input variant Defines the style variant of the toggle button, using values from the ToggleVariants enum.
Input left Configuration for the left graphic (icon or image) in the toggle button, using the ToggleGraphics interface.
Input right Configuration for the right graphic (icon or image) in the toggle button, using the ToggleGraphics interface.
Input thumbImg Specifies an image to display on the thumb or knob of the toggle button.
Output toggleStateChange Emits an event whenever the state of the toggle button changes, passing the new state (true or false).
Public _toggleState Stores the current internal state of the toggle button (true or false).
Public currentToggleStatus Keeps track of the toggle's animation status using the ToggleStatuses enum.
Public circleMask An object containing unique identifiers for the toggle's SVG masks to differentiate multiple toggle instances.
