Popup Component

A compact and interactive chip for displaying and interacting with small pieces of information.
Demo
Open Modal
Required Imports
import { LibModalModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description
Input isOpen Controls whether the modal is open or closed.
Input icon The icon displayed in the modal.
Input title The title text of the modal.
Input message The main message or content of the modal.
Input extendedMessage An optional extended message for additional information.
Input cancelButton Text for the cancel button in the modal.
Input confirmButton Text for the confirm button in the modal.
Input checkboxText Text displayed for the optional checkbox within the modal.
Input progBarCurrentStep Current step value for a progress bar, if used.
Input progBarTotalSteps Total steps for a progress bar, if used.
Input customClass Custom class applied to the modal for styling.
Input backdropDismiss Determines if the modal can be dismissed by clicking on the backdrop.
Input modalComponent Component to dynamically load into the modal.
Input modalComponentData Data passed to the dynamically loaded modal component.
Output modalConfirm Emits when the confirm button is clicked.
Output modalDismiss Emits when the modal is dismissed or closed.
Output afterCloseEvent Emits after the modal is closed.
