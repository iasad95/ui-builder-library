Phone Number

A specialized input used for entering phone numbers.
Demo
{{ phoneForm.controls.phone.value | json }}
Required Imports
import { TelInputModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description
Input value The current phone number value entered in the input field.
Input enablePlaceholder Determines whether the placeholder should be displayed.
Input maskPhoneNumber Enables masking of the phone number as it's entered.
Input eagerMask Determines if the mask should be applied eagerly.
Input customPlaceholder A custom placeholder for the input field.
Input numberFormat Format for displaying the phone number, e.g., International or National.
Input cssClass The CSS class applied to the input field.
Input onlyCountries Limits the countries available for selection.
Input searchCountryFlag Enables the country flag search in the dropdown.
Input searchCountryField Specifies which fields to use for searching the country, e.g., name or dial code.
Input searchCountryPlaceholder Placeholder text for the country search field.
Input maxLength The maximum length allowed for the phone number input.
Input selectFirstCountry Automatically selects the first country in the list.
Input selectedCountryISO Sets the default selected country by its ISO code.
Input phoneValidation Enables phone number validation.
Input inputId The ID attribute for the phone input field.
Input separateDialCode Displays the dial code separately from the phone number.
Input showDropDownArrow Determines if the dropdown arrow should be displayed for the country selection.
Input icons Custom icons for the input field.
Output countryChange Emits the selected country whenever it changes.
Options
Description Option
Toggle country selection input.
Mark the input field as touched.
Write string.
Reset the control value.
