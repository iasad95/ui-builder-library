List

A concise list component for organizing and displaying information in a clear and structured manner.
Demo
Required Imports
import { ListModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description Default
Input listItems An array of ListItemModel objects representing the items to display in the list. n/a
Input hideMenuIcon If true, the menu icon for each list item will be hidden. false
Input listTitle The title of the list displayed at the top. n/a
Input disableTranslation If true, disables translation for the list items. false
