TextList
TextList is a generic text list that the user can add to/delete from
Demo
Required Imports
import { TextListModule } from 'ui-builder-component-lib';
Properties
Type Name Description Data type Required Default Demo
Input placeholderText Sets the default placeholder text for the 'add-item' input string true n/a
Input textListStrings An array of strings representing the list string[] true n/a n/a
Input isUrlList A flag to specify whether the list items should be treated as URLs.
Input disabled A flag to disable the text input and list modification functionalities.
Output listChange An event emitter that outputs the updated list of text items when changes are made.
Public buttonsHidden A boolean to control the visibility of action buttons.
Public showIndex An index that tracks the currently active or visible item in the list.
Public textValue The value currently entered in the input field, used for adding new items.
