Checkbox

A checkbox that can be used in a selection or toggle.
Demo
Required Imports
import { DashedCheckboxModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description default
Input checked Determines whether the checkbox is checked. false
Input disabled Determines whether the checkbox is disabled. false
Output toggle Emits an event when the checkbox is toggled, passing the new checked state. n/a
CSS Properties
Name description default
-color defines the color of the checkmark and the border black
--check-line-width defines how thick the check line should be 3px
--size defines how wide and high the checkbox should be. The checkbox is always squared shape 32px
