Input config The configuration object for the toaster, which defines its behavior, icon, life duration, and appearance.
Output closeClick Emits an event when the close button is clicked on the toaster.
Output checkboxChange Emits an event when the checkbox in the toaster is toggled.
Public toasterLife The duration for which the toaster stays visible, defined by the config or defaulted to 2000ms.
Public show A boolean flag indicating whether the toaster is currently visible or not.
Public checkbox A boolean indicating whether the toaster's checkbox is checked.
Public toasterIcon The name of the icon to display in the toaster, based on its severity.
