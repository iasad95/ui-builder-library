JSON Editor

A concise JSON editor component for easy manipulation and visualization of JSON data in your application.
Demo
Required Imports
import { LibJsonEditorModule } from 'ui-builder-component-lib';
Example usage
{{ jsonEditorTemplate }}
Properties
Type Name Description Default
Input data The data you want to display. n/a
Input expandAll Expand/collapse JSON data, with a value of true/false. n/a
Input language The language to display the JSON data, e.g., 'en', 'es', 'fr'. n/a
Input mode Set the mode to 'edit' or 'view'. n/a
Input search If true, a search bar will be displayed in the editor. false
Input mainMenuBar If true, the main menu bar (for features like undo, redo, etc.) will be shown. false
