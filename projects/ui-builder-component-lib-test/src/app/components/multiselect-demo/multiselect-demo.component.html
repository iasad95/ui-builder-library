Multiselect
ExpansionPanel is a wrapper around Angular Material's MatExpansionPanel with a standardized UI Builder look and feel. It supports a conventional expansion behavior as well as a "flyout" style menu. When there are more than 50 items in the list it will switch to search mode with allows the user to filter options based on a non-case sensitive includes on the select option label.

Properties
Type Name Description Data type Required Default Demo
Input dialogTitle The title of the modal that opens when the Add button is clicked. string true ''
Input options The available options to select from. SelectOption[] true n/a n/a
Input selectedOptions The selected options to select from. SelectOption[] false n/a n/a
Input submitted Tells the MultiSelect if the form has been submitted. If true, and if required is also true, the UI is updated to indicate the error and display an error message. boolean false false
Input required Tells the MultiSelect if this is a required field. If true, an indicator displays next to the label. boolean false false
Input disabled Tells the MultiSelect if this is a disabled field. boolean false false
Input showDialogMessage Toggle show/hide of dialog message. string true true
Output selectedOptionsChange Emits an event indicating that the selected options have changed and containing the new set of selected options. SelectOption[] n/a n/a n/a
Input optionGroups An array of option groups, each containing a group name and an array of options of type SelectOption.
Input fieldName The name of the field for reference in forms and events.
Input showRequiredIndicator Boolean to display an asterisk (*) if the field is required.
Input optionDeleteConfirmationObsv An observable for confirming the deletion of an option.
Input headerIconTray An array of ItemActionHandler objects for handling icon actions in the dialog header.
Output optionDelete An event emitted when an option is deleted, containing the field name and the deleted option.
