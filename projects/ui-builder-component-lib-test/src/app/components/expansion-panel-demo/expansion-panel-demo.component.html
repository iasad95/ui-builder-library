Required Imports
import { ExpansionPanelModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description Default
Input title The title displayed in the expansion panel header. n/a
Input expanded Controls whether the panel is expanded or collapsed by default. true
Input showToggleText If true, prevents the panel from toggling when the header is clicked. false
Input descriptionText An optional description text displayed within the panel content. n/a
Output expandedChange Emits an event when the panel's expanded/collapsed state changes. n/a
