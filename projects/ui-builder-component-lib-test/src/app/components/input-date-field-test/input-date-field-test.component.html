Date Field

A user-friendly date input for easy selection in the app.
Demo
Required Imports
import { InputDateFieldModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description Default
Input label The label displayed for the input date field. n/a
Input defaultDate The default date to display in the calendar. n/a
Input maxDate The maximum selectable date in the calendar. n/a
Input minDate The minimum selectable date in the calendar. n/a
Input showTime Indicates whether to show the time picker. false
Input hieActions Enables HIE-specific actions in the date picker. false
Input readonly Specifies if the input field is read-only. false
Input defaultDayHighlight Highlights the default day in the calendar if set. false
Input showIcon Determines if a calendar icon should be shown next to the input field. n/a
Input timeOnly Indicates if the picker should only display time. false
Input view Specifies the view type of the calendar. date
Input disabled Specifies whether the input field is disabled. false
Input dateFormat The format for displaying dates. mm-dd-yy
Input placeholder The placeholder text for the input field. n/a
Input appendTo Specifies where to append the date picker popup. Can be a string, template, or element reference. n/a
Input stepMinute The step interval for minutes in the time picker. 15
Input mask The input mask for date or time formatting. n/a
Input styleClass CSS classes to apply to the input field for styling. n/a
Input timeFormat The format for displaying time. hh:mm
Output focusEvent Emits an event when the input field is focused or loses focus. n/a
