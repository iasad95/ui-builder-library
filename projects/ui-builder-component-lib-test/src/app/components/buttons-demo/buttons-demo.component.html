Input and Output Properties
Type Name Description
Input hoverOutline Determines if the button should display a hover outline effect.
Input disabled Disables the button if set to true.
Input type Specifies the button type attribute (e.g., "button" or "submit").
Input form The form attribute for associating the button with a form.
Input label The text label displayed on the button.
Input showArrowIcon Determines if an arrow icon should be displayed on the button.
Input disableTextShadow Disables the text shadow effect if set to true.
Input notificationCount Displays a notification badge with the provided count.
Input loading$ An observable that controls the button's loading state.
Input illuminateIcon Determines if the button's icon should be illuminated.
