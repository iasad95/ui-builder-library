Bottom Modal

A popup from the bottom of the screen that shows extra information or choices, making it easy for users to interact and navigate.

Input and Output Properties
Type Name Description Default
Input breakpoints An array of breakpoints that define the modal's snap points. 0, 0.25, 0.5, 0.75
Input initialBreakpoint The initial breakpoint where the modal should open. 0
Input isOpen Controls whether the modal is open or closed. When set, it triggers modal visibility. n/a
Input canDismiss Determines if the modal can be dismissed. true
Input showArrow Shows or hides an arrow in the modal header. true
Input showBackBtn Displays a back button in the modal if set to true. n/a
Input loading Controls a loading state within the modal. n/a
Input showHeaderSeparator Shows or hides a separator in the header. true
Input title Defines the title text of the modal. n/a
Input description Sets a description for the modal. n/a
Input submitButtonTitle Defines the text for the submit button. n/a
Input cancelButtonTitle Defines the text for the cancel button. n/a
Input validModal Indicates if the modal is valid for submission. true
Input adaptToContentHeight If true, the modal adjusts its height based on content. true
Input customClass Applies a custom CSS class to the modal for styling. n/a
Output isOpenChange Event emitter that emits when the modal's open state changes. n/a
Output breakpointChange Emits the current breakpoint when it changes. n/a
Output closeModal Emits when the modal is closed. n/a
Output submitModal Emits when the submit button is clicked. n/a
Output backClick Emits when the back button is clicked. n/a
