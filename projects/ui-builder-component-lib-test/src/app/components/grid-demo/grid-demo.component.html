Required Imports
import { GridModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description Default
Input enableSelection Enables row selection functionality. false
Input enableGridFilters Enables the use of grid filters. false
Input enableCreate Enables the "Create" action in the grid. false
Input enableRefresh Enables the refresh button in the grid. false
Input enableTrash Enables the trash icon in the grid for deletion functionality. false
Input isLoading Indicates whether the grid is loading data. false
Input totalRecords Total number of records in the grid. 0
Input rowsPerPage Number of rows per page in the grid. 10
Input first Index of the first record to display. 0
Input rowsPerPageOptions Options for rows per page. [10, 25, 50]
Input paginatorDataTypeLabel Label for paginator data type. Records
Input selectAllConfirmButtonLabel Label for "Select All" confirm button. n/a
Input selectAllConfirmMsg Message displayed when "Select All" is confirmed. n/a
Input currencyDisplayCode Currency display code. USD
Input minimumColumnWidth Minimum width for columns. -1
Input sortField Field used for sorting in the grid. n/a
Input groupByField Field used for grouping the grid. n/a
Input gridFilterConfig Configuration for grid filters. n/a
Input filterDefinitions Definitions for grid filters. n/a
Input userFiltersList List of user filters applied to the grid. n/a
Input dateFormat Date format used in the grid. n/a
Output lazyLoad Event emitted for lazy loading of grid data. n/a
Output columnsChange Event emitted when columns are changed. n/a
Output editComplete Event emitted when an edit is completed in the grid. n/a
Output refreshData Event emitted to refresh the grid data. n/a
