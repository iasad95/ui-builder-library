Select Field

A choice picker for easy user selection.
Demo
Required Imports
import { InputFieldSelectModule } from 'ui-builder-component-lib';
Input and Output Properties
Type Name Description
Input multiple Enables multi-selection if set to true.
Input label The label text displayed above the input field.
Input placeholder Placeholder text displayed inside the input field.
Input optionLabel Property of the option used to display the label in the dropdown.
Input optionValue Property of the option used as the value for the form control.
Input options An array of options available for selection.
Input optionsTemplates An array of templates to customize the option display.
Input filterBy Property used for filtering the options.
Input enableFilter If true, enables the filter functionality for the options.
Input readonly Sets the input field to readonly mode.
Input disabled Disables the input field if set to true.
Input showClear Displays a clear button to clear the selection.
Input editable If true, makes the input field editable.
Input withTemplate Determines if custom templates are used for displaying options.
Input enableTranslation If true, enables translation support for the options and label.
