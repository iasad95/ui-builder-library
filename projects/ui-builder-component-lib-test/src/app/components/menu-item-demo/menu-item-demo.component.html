LibMenuItem
LibMenuItem is a simple generic class holding a label and a generically typed value.
Properties
Name Description Data type
templateRef A reference to a template that can be used to render custom content for the MenuItem. TemplateRef<HTMLElement>
title The title text displayed for the MenuItem. string
titlePrefix Additional text that is prefixed to the title of the MenuItem. string
titleSuffix Additional text that is suffixed to the title of the MenuItem. string
iconUrl The URL of an icon image to be displayed alongside the MenuItem. string
iconName The name of an icon to be displayed alongside the MenuItem (alternative to iconUrl). string
cssClass A CSS class to be applied to the MenuItem for styling purposes. string
active A boolean indicating whether the MenuItem is currently active. boolean
disabled A boolean indicating whether the MenuItem is disabled. boolean
separator A boolean indicating whether the MenuItem is a separator, used to visually separate groups of items. boolean
subMenu An array of LibMenuItem objects representing a sub-menu associated with the MenuItem. LibMenuItem[]
listItem An unknown type representing additional data associated with the MenuItem. unknown
listItemIndex An index indicating the position of the MenuItem in a list. number
command A callback function to be executed when the MenuItem is triggered. (event?: LibMenuItem) => void
