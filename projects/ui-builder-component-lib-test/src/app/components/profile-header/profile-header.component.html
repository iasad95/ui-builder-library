Input photo URL or path to the profile photo image.
Input name The name of the person displayed on the profile header.
Input leftIcon Icon displayed on the left side of the header.
Input rightIcon Icon displayed on the right side of the header.
Input leftText Text displayed alongside the left icon.
Input rightText Text displayed alongside the right icon.
Input orientation Orientation of the profile header, defaults to ProfileHeaderOrientation.horizontal.
