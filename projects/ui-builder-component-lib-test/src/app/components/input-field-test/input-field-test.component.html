Input and Output Properties
Type Name Description Default
Input name The name attribute for the input field. n/a
Input autocomplete Autocomplete attribute for the input field. n/a
Input label The label for the input field. n/a
Input placeholder Placeholder text for the input field. n/a
Input type The type of the input field (e.g., 'text', 'email'). n/a
Input readonly Specifies if the input field is read-only. false
Input focus Determines if the input field should be focused on load. false
Input enableGoogleAddress Enables Google address autocomplete for the input field. false
Input locationAutocompleteConfig Configuration object for Google Maps location autocomplete. n/a
Input defaultLocation Coordinates for the default location (latitude and longitude). n/a
Input mask Mask for formatting the input value. n/a
Input prefix Prefix displayed before the input value. n/a
Input suffix Suffix displayed after the input value. n/a
Input maxLength The maximum number of characters allowed in the input field. 20
Input value The value of the input field. n/a
Input address Address used when Google address autocomplete is enabled. n/a
Input textarea Determines if the input field is a textarea. false
Input resize Specifies if the textarea is resizable. false
Input patterns Defines custom patterns for input masking. n/a
Input trimWhiteSpace Specifies if whitespace should be trimmed from the input. true
Output locationChange Emits an event when the location is changed using Google address autocomplete. n/a
Output keyDown Emits an event when a key is pressed in the input field. n/a
Output onBlur Emits an event when the input field loses focus. n/a
