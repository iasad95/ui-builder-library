InfoCard
InfoCard is a flexible component designed to display and interact with an ItemSummary. It displays the ItemSummary's abbreviation, icon, or image (as appropriate), the header, and any subheaders and detail lines. By customizing the second and third arguments of any header, subheader, or detail, the consumer can direct the InfoCard to display tooltips or run relevant functions. The InfoCard is reactive to whether or not the ItemSummary is marked as disabled, and allows for generic execution of the ItemSummary's actionHandler. Additionally, it provides an optional checkbox allowing each ItemSummary to be selected or deselected e.g. as part of a list.
