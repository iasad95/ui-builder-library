Tooltip

The Tooltip is a directive that adds a message to a Tooltip component and binds it to the Angular CDK.

Required Imports
import { TooltipModule } from 'ui-builder-component-lib';
Properties
Type Name Description Data type Required Default Demo
Input libTooltip Sets the tooltip content. SafeHTML true n/a n/a
Input delayTime Sets time to wait to display tooltip in milliseconds. number false 1000
Input directionPreference Where to display tooltip in reference to the target TooltipDirectionPreference false TooltipDirectionPreference.Bottom
Input title Tooltip title SafeHTML false null n/a
Input hideIcon hide tooltip icon boolean false false n/a
Public mouseEnter() Event listener for mouse enter, triggers the tooltip to be shown after a delay.
Public mouseOut() Event listener for mouse leave, hides the tooltip.
Public onClick() Event listener for mouse click, hides the tooltip when clicked.
